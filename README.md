# ReportApp

Учебное консольное приложение на C# для построения разузловки ДСЕ и выгрузки результата в Excel.

## Что делает программа

Приложение позволяет:

- просматривать список ДСЕ
- просматривать все элементы из базы данных
- строить разузловку по `Id`
- выгружать разузловку в Excel

## Что используется

- C#
- .NET 8
- Entity Framework Core 8
- Microsoft SQL Server
- ClosedXML

## Структура данных

В базе хранится таблица `Items`, где для каждого элемента задаются:

- название
- тип (`ДСЕ` или деталь)
- партия
- родительская партия
- количество
- порядок вывода
- комментарий

Связь между элементами строится через поля:

- `Batch` — партия самого элемента
- `ParentBatch` — партия родительской ДСЕ

## Как работает разузловка

Программа находит выбранный элемент, затем рекурсивно получает все вложенные элементы по полю `ParentBatch`.  
Результат можно вывести в консоль или сохранить в Excel.

## Как запустить

1. Открыть проект в Visual Studio 2022
2. Проверить строку подключения к SQL Server в `AppDbContext`
3. Выполнить миграции базы данных
4. Запустить проект

## Возможности меню

- показать все ДСЕ
- показать все элементы
- построить разузловку по `Id`
- выгрузить разузловку в Excel

Проект сделан в учебных целях для практики работы с:
